# TruthLens

TruthLens is an AI-powered News Aggregator and Credibility Analysis Platform built using the MERN stack.

## Features
- Live news feed
- Search & category filtering
- Article details page
- Save/bookmark articles
- User authentication
- MongoDB storage

## Tech Stack
- React + Vite + Tailwind CSS
- Node.js + Express
- MongoDB Atlas
- JWT Authentication
- NewsData.io API

- Full app deployed on Render
